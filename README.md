# Lean-Game-Server---Robo-Scribble-solutions

Hello there!!! I'm an undergrad in their second year who had just finished playing the Robo/Scribble game in the Lean Game Server!!! One of the most frustrating aspects of attempting this particular game was finding the solutions. In some cases, solutions from the dev's side (the game dev's) were only possible in lean 3, or tactics disallowed in the current version of the game. I have somehow managed to maintain most of the code, either on my own, or with a little help of the developer's solutions, to make a solution key exclusively in lean 4 and using tactics present in the game.

So here is a repository of those solutions. Please enjoy!!! If you find any issues or feel something is handwavy, feel free to file a PR!!

These solutions were made from 04/06/2026 to 27/06/2026.
